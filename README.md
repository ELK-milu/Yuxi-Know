## 📖 简介

语析是一个功能强大的智能问答平台，融合了 RAG 知识库与知识图谱技术，基于 LangGraph + Vue.js + FastAPI + LightRAG 架构构建。

详细文档请查看文档中心：[https://xerrors.github.io/Yuxi-Know/](https://xerrors.github.io/Yuxi-Know/)
